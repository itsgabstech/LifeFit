# Fitness App - Projeto em Grupo

## Estrutura do Projeto

Dividido em 3 PRs independentes para 3 pessoas:

### PR 1 - PESSOA 1: Autenticação
- `components/login.tsx` - Página de login
- `components/header.tsx` - Header com nome e logout
- Arquivo: `PR1_AUTENTICACAO.md`

### PR 2 - PESSOA 2: Dashboard
- `components/exercise-card.tsx` - Cards de exercícios
- `components/daily-plan.tsx` - Plano do dia
- Arquivo: `PR2_DASHBOARD.md`

### PR 3 - PESSOA 3: Navegação e Páginas
- `components/navigation.tsx` - Menu inferior
- `components/pages/` - Novas páginas
- Arquivo: `PR3_NAVEGACAO.md`

## Como Começar

1. Clone o repositório
2. `npm install` ou `pnpm install`
3. `npm run dev`
4. Acesse `http://localhost:3000`

## Stack
- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
